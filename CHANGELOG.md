## [1.0.0] - 2024-11-26

### Added
- Initial release of Terrorist Network Disruption using Temporal GNN
- Advanced Temporal GNN architecture with:
  - Multi-head temporal attention
  - Memory-augmented networks
  - Graph transformer layers
  - Adaptive time encoding
  - Causal temporal convolution
- Multi-layer network support (5 layers):
  - Physical network
  - Digital network
  - Financial network
  - Ideological network
  - Operational network
- Critical node detection with ensemble methods
- Temporal resilience prediction
- Adversarial robustness analysis
- Self-supervised learning framework:
  - Temporal link prediction
  - Contrastive learning
  - Node reconstruction
- Comprehensive baseline comparisons:
  - Static GNN methods (GCN, GAT, GraphSAGE)
  - Traditional centrality measures
  - Simple temporal GNN
- Statistical significance testing
- Ablation study framework
- Synthetic terrorist network generation
- Complete experimental pipeline
- Documentation and examples

### Features
- Command-line interface for experiments
- Google Colab notebook support
- GPU acceleration support
- Extensive test suite
- Publication-ready visualizations

### Research Contributions
- Novel T-GNN architecture for network disruption

- Comprehensive evaluation framework
