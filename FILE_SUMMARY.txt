=============================================================================
사용 방법
=============================================================================

1. 압축 해제:
   tar -xzf terrorist-network-tgnn-v1.0.0.tar.gz
   cd terrorist-network-tgnn

2. 의존성 설치:
   pip install -r requirements.txt

3. 빠른 실행:
   - Jupyter 노트북: examples/terrorist_network_gnn_demo.ipynb
   - Python 스크립트: python src/main_experiment.py
   - 커맨드 라인: python src/main_experiment.py --mode demo

4. 전체 문서는 README.md 참조

=============================================================================
